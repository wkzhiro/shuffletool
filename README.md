# カテゴリ優先度付きシャッフルツール

## 概要
複数のカテゴリ情報を持つユーザーデータを、カテゴリの優先度に基づいて最適なグループに分割するStreamlitアプリケーションです。

## 機能
- Excelファイルのアップロードとデータ読み込み
- 3つのカテゴリによるグループ分け（カテゴリ1、カテゴリ2、カテゴリ3）
- カテゴリの優先順位設定（5点、3点、1点の配点）
- 指定サイズでのグループ分割とシャッフル
- 最高スコアでのグループ割り当て結果をCSV出力

## 使用方法
1. Excelファイルを準備
   - 1列目：ユーザーID
   - 2列目：カテゴリ1
   - 3列目：カテゴリ2（数値）
   - 4列目：カテゴリ3

2. アプリケーションの起動
   ```bash
   streamlit run main.py
   ```

3. Webインターフェースでの操作
   - Excelファイルをアップロード
   - 各カテゴリのグループ分けを設定
   - カテゴリの優先順位を設定
   - グループサイズとシャッフル回数を設定
   - シャッフル実行

## ファイル構成
- `main.py` - メインアプリケーション
- `requirements.txt` - 必要なPythonパッケージ
- `test.xlsx` - テスト用データファイル（17行4列）
- `myenv/` - 仮想環境（venv）

## 必要なパッケージ
- streamlit==1.41.1
- openpyxl==3.1.5
- pandas
- numpy

## インストール
```bash
pip install -r requirements.txt
```

## テストデータ
`test.xlsx`には17人のユーザーデータが含まれており、以下の構造になっています：
- ユーザーID（1-17）
- カテゴリ1（文字列データ）
- カテゴリ2（数値データ）
- カテゴリ3（文字列データ）

## 動作原理
1. 各カテゴリでユニークな値をグループ分け
2. 優先順位に応じてスコア配点（1位：5点、2位：3点、3位：1点）
3. 指定サイズでグループを作成し、同じカテゴリグループ内のメンバーが多いほど高スコア
4. 9点満点（3カテゴリ全一致）のグループは自動固定
5. 最高スコアの組み合わせを保存してCSV出力
